# Traffic light optimizer

Ένα λογισμικό εργαλείο για τον σχεδιασμό και τη βελτιστοποίηση φωτεινής σηματοδότησης ισόπεδων κόμβων. Το πρόγραμμα υπολογίζει τη βέλτιστη σηματοδότηση βάσει κυκλοφοριακών φόρτων, αξιολογεί τη στάθμη εξυπηρέτησης (LOS) και πραγματοποιεί προβλέψεις μελλοντικού φόρτου.

## Κύριες Λειτουργίες
- **Αυτοματοποιημένος Υπολογισμός:** Υπολογισμός ροών κορεσμού και καθυστερήσεων βάσει μοντέλων συγκοινωνιακής μηχανικής.
- **Optimization Engine:** Εξαντλητική αναζήτηση για τον εντοπισμό του βέλτιστου χρόνου κύκλου.
- **Data Validation:** Έλεγχος εγκυρότητας δεδομένων εισόδου και υποστήριξη αρχείων .dat.
- **Visualization:** Παραγωγή γραφημάτων σηματοδότησης και πρόβλεψης κυκλοφοριακού φόρτου.

## Τεχνικά Χαρακτηριστικά
- **GUI:** Υλοποιημένο με `tkinter` για ευκολία στη χρήση.
- **Computational Core:** Χρήση `numpy` για πολυωνυμικές προβλέψεις και μαθηματική μοντελοποίηση.
- **Architecture:** Modular δομή με διαχωρισμό λογικής (logic), μοντέλων (models) και διεπαφής (gui).
